# Pokedex

## Setup instruction

  1. Clone the project <br><br>
      ```  
      https://github.com/chintu46306/pokedex.git 
      
      ```

    
 2. Move into the directory <br><br>
    ```
    cd pokedex

    ```
3. install dependencies   <br><br>
    ```
    npm i

    ```
4. run the server
    ```
    npm run dev
  
    ```
